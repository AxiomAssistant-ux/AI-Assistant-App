This folder contains app assets.

Required image files (you can replace with your own):
- icon.png (1024x1024) - App icon
- splash.png (1284x2778) - Splash screen image
- adaptive-icon.png (1024x1024) - Android adaptive icon
- favicon.png (48x48) - Web favicon

For development, the app will work without these images.
You can generate placeholder images using any image editing tool.
